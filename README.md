<h1 align="center">Pathfinding algorithms animated</h1>

## Definition
A collection of pathfinding algorithms animated on a random maze

## Features
- Visited nodes
- Nodes to be visited
- Path found

## Algorithms
- A*
- Djikstra
- DFS
